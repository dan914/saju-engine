| year | **연주** | 국어 음운 규칙 준수 |
| confidence | **높음** (별도 validity 필드 분리) | 성격/파격과 명확히 구분 |

## 3. 중요 권고사항 (전문가 피드백)

### ⚠️ Critical Warnings

#### 1. Confidence vs 성격/파격 분리 필수
```json
// ❌ 잘못된 사용
{
  "structure": {
    "primary": "정관격",
    "confidence": "성격"  // 잘못된 매핑!
  }
}

// ✅ 올바른 사용
{
  "structure": {
    "primary": "정관격",
    "confidence": "높음",      // UI 신뢰도 (확률)
    "validity": "성격"         // 격국 성립 상태 (판정)
  }
}
```

**이유**:
- confidence = 통계적 확신도
- 성격/파격 = 격국 규칙 충족 여부
- 의미 축이 다름 → 1:1 매핑 시 오해 발생

#### 2. 연주 vs 년주
- **표준 표기**: 연주(年柱)
- **허용 별칭**: 년주
- **권장**: 문서·UI는 '연주' 사용

#### 3. 반합 vs 방합
- **반합(半合)**: 삼합의 절반 (2개 지지만 만남)
- **방합(方合)**: 방위별 합 (별개 개념)
- **주의**: banhe → 반합 (방합 아님!)

   - "년주" = 관용 허용

4. **동음이의어 주의**
